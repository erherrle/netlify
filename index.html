Let's Git Started on Git Hub

If you've managed to find your way to GitHub, you would have been greeted by this cheeky little cat. If you see him, you're in the right place.

Let's walk through the steps of creating your first test repository.

Here is what you will need to get started:

  • A GitHub Account
  • Terminal
  • Sublime Text

    • Creating your Repository

    Once you have logged into your GitHub account, you should click the green button that says "New" at the top left corner of your browser in the section titled "Repositories".

    Once you have named and created your repository, this will prompt step by step instructions to follow when setting up your repository in Terminal.

    Create a folder on your desktop with the same name as your repository to store your files.

    Setting Things up in Terminal

    This is definitely the most difficult part, but hang tight and pay attention to details.

    Begin by establishing your location in terminal. To do this type "cd" press space and then drag the file you just created for your repository from your desktop and into the Terminal window. Press enter.

    Next you will type the following commands provided by GitHub into terminal:

    1. echo "# testforproject" >> README.md
    2. git init
    3. git add README.md
    4. git commit -m "first commit"
    5. git branch -M main
    6. git remote add origin (COPY AND PASTE LINK FOR YOUR SITE - EX: git add origin https://github.com/erherrle/testforproject.git)
    7. git push -u origin main

    Tip: Press "return" after each step.

    Start Creating Your Site

    Open Sublime Text and create two new files one titled "index.html" and the other "style.css". We will then add both files to your repository's folder on your desktop and push them to GitHub.

    To do this type the following commands into Terminal:

    1. git add index.html
    2. git commit -am "added index.html"
    3. git push origin main

    Do the same for your style file.

    Add Content to your Site

    Now you can begin coding in html on Sublime and add content to your site.

    Don't forget to save frequently and push your changes to GitHub by following the three steps above.

    Use the command "git status" to make sure you have commited everything and "your working tree is clean.
